My Projects

Most of my projects sit at the intersection of data engineering, web development, and data visualization. A lot of them started as coursework at Northeastern, but each one pushed me to learn something new — whether that meant figuring out Docker networking, designing a REST API from scratch, or making an Altair visualization interactive.

I’m especially drawn to projects that work with real, messy data and turn it into something clear or useful. I naturally gravitate toward backend and data infrastructure work, but I’ve also been building my frontend skills. For example, in my work on the Good Dog Licensing website, I’ve been translating Figma designs into React and Tailwind components and learning more about how the frontend and backend pieces fit together.

I also really enjoy collaborative projects. Through group work, I’ve gotten comfortable using Git in team environments, working with branches, resolving merge conflicts, and coordinating features across different parts of a code base. Each project below taught me something different and helped me build a broader toolkit as a developer.

Going forward, I’m excited to keep building more end-to-end projects that connect a data layer to a real user interface. I’m also always trying to get better at writing clean, readable code that other people can easily understand and build on.
