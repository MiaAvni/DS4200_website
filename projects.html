My Projects

My projects span data engineering, full-stack web development, and data visualization. Most started as coursework at Northeastern but pushed me to learn something genuinely new each time — whether that was Docker networking, designing a REST API from scratch, or making an Altair chart interactive.

I'm especially drawn to projects that handle real, messy data and produce something useful or clear. I tend to gravitate toward backend and data infrastructure work, but I've been building frontend skills too through my work on Good Dog Licensing's website, where I convert Figma designs into React and Tailwind components.

I enjoy working collaboratively and have experience with Git workflows in team settings, resolving merge conflicts, and dividing up features across branches. Each project below taught me something different and I'm always looking for the next interesting problem to work on.

Going forward I want to keep building end-to-end projects that connect a data layer to a real user interface, and to keep getting better at writing clean, well-documented code that others can read and build on.
